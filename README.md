### Install Project
`git clone https://github.com/ploynapatbunsena/Swensens.git`
or
`git clone git@github.com:ploynapatbunsena/Swensens.git`

### Start Backend 
`node .\index.js`
port http://localhost:4000/

### Start Admin
`npm run dev`
port http://localhost:5173/

### Start Frontend
`npm strat`
port http://localhost:3000/
